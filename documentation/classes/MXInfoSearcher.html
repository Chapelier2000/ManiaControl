MXInfoSearcher

Implements \Iterator, \Countable

MXInfoSearcher - Search info for TM2/SM/QM maps from ManiaExchange Created by Xymph <tm@gamers.org> based on: http://api.mania-exchange.com/ http://tm.mania-exchange.com/api http://sm.mania-exchange.com/api http://tm.mania-exchange.com/threads/view/218 Derived from TMXInfoSearcher

v1.6: Added Countable interface to searcher class v1.5: Added MXInfo $titlepack (TM2/SM); add support for environment matching v1.4: Fixed an error checking bug v1.3: Added MXInfo $maptype (TM2/SM) v1.2: Updated to use MX API v2.0 and add/fix support for SM; added MXInfo

  $trkvalue (TM2, equals deprecated $lbrating), $unlimiter (TM2/SM),

Searches MX for maps matching name, author and/or environment; or search MX for the 10 most recent maps

__construct(String $game, String $name, String $author, String $env, Boolean $recent) : \MXInfoSearcher

Arguments

$game

String

MX section for 'TM2', 'SM', 'QM'

$name

String

The map name to search for (partial, case-insensitive match)

$author

String

The map author to search for (partial, case-insensitive match)

$env

String

The environment to search for (exact case-sensitive match from:

   TMCanyon, TMStadium, TMValley, SMStorm, ...)

$recent

Boolean

If true, ignore search parameters and just return 10 newest maps

   (max. one per author)

Response

\MXInfoSearcher

If ->valid() is false, no matching map was found;

   otherwise, an iterator of MXInfo objects for a 'foreach' loop.
   Returns at most 100 maps ($maxpage * 20).
